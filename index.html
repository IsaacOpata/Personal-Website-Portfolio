Travel System

The Travel System in C++ is a program that allows users to input their preferred location, airline, number of travelers, and airfare to obtain a comprehensive breakdown of the costs of their trip for the entire family. It provides the carrier's name, airfare, round-trip ticket price for adults, and round-trip ticket price for underage travelers under the age of 18 with a 25% discount. The program is implemented using linked lists, nodes, and string data types to ensure efficient processing of data with minimal code.

Chair Sales Tracker

The Chair Sales Tracker is a program developed in C++ that tracks the amount of chairs sold for three chair styles: American Colonial, Modern, and French Classical. It calculates and outputs the total dollar sales for each style and overall sales in fixed point notation with two decimal places. The program ensures efficiency and accuracy through the use of linked lists, nodes, and string data types. It's a user-friendly tool for furniture stores and manufacturers to track their sales and make informed decisions for future sales and production.
